# Implementation Plan: 004 — AI Chatbot for Task Management

**Branch**: `main` | **Date**: 2026-02-28 | **Spec**: [spec.md](./spec.md)

---

## Summary

Add an AI-powered chatbot to Taskify. Users type natural language messages to manage tasks. The backend extends the existing FastAPI app with: two new DB tables (conversation, message), one new chat route (`POST /api/{user_id}/chat`) streaming SSE responses, a new MCP server (5 tools: add/list/complete/delete/update task), and the OpenAI Agents SDK configured to use Groq (free tier). The frontend adds a `/chat` page using OpenAI ChatKit, auth-gated, matching the existing dark theme. All state persists to PostgreSQL — the server holds no memory between requests.

---

## Phase 0: Research Findings (Complete)

All decisions resolved in `research.md`. Key conclusions:

1. **Groq** replaces OpenAI — same `openai-agents` SDK, different `base_url` + `model`
2. **MCP stdio transport** — subprocess spawned by Agents SDK, no separate deployment
3. **SSE via `sse-starlette`** — `EventSourceResponse` for FastAPI streaming
4. **History window**: last 20 messages per conversation → passed to agent as input
5. **New conversation auto-load**: GET `/conversations/latest` on page open (Option A from clarify)
6. **AI failure UX**: friendly error message in chat stream (Option A from clarify)
7. **user_id in system prompt**: prevents agent from using wrong user_id in tool calls

#### Chat Endpoint (`backend/routes/chat.py`)
```python
from agents import Agent, Runner, MCPServerStdio
from sse_starlette.sse import EventSourceResponse

@router.post("/{user_id}/chat")
async def chat(user_id: int, req: ChatRequest, current_user=Depends(get_current_user)):
    if current_user.id != user_id:
        raise HTTPException(403)

    async def event_stream():
        conv = load_or_create_conversation(user_id, req.conversation_id)
        history = load_history(conv.id)
        save_message(conv.id, user_id, "user", req.message)

        async with MCPServerStdio(params={"command": "python", "args": ["backend/mcp_server.py"]}) as mcp:
            agent = Agent(
                name="taskify",
                model="llama-3.3-70b-versatile",
                instructions=SYSTEM_PROMPT.format(user_id=user_id),
                mcp_servers=[mcp]
            )
            result = await Runner.run(agent, history + [req.message])
            response_text = result.final_output
            tool_calls = extract_tool_calls(result)

        save_message(conv.id, user_id, "assistant", response_text)
        yield {"data": json.dumps({"conversation_id": conv.id,
                                    "response": response_text,
                                    "tool_calls": tool_calls,
                                    "type": "done"})}

    return EventSourceResponse(event_stream())
```
